# INST346 — Computer Networking Fundamentals

Repository containing coursework, Packet Tracer simulations, networking labs, lecture materials, and projects completed throughout INST346.

## Overview

This course introduced fundamental networking concepts, network architecture, routing, switching, network design, and practical network implementation using Cisco Packet Tracer.
